dasBlog Configuration Error

Summary

An error has been encountered while writing to your content folder. Most likely this error is caused by a configuration error on your server. Contact your hosting provider or check your permissions for

  • SiteConfig
  • content
  • logs

And ensure that the accounts below have Read/Write access to the above folders:

Windows 2000/XP

  • ASP.NET Machine Account "{machinename}\ASPNET"

Windows Server 2003

  • ASP.NET Machine Account "{machinename}\ASPNET"
  • Network Service "NT AUTHORITY\NETWORK SERVICE"

For more information see:

http://dasblog.info/SetupInstall.aspx

Web Hosting Help

If DasBlog is being hosted on an ASP.NET hosting provider, send them the following information to configure DasBlog correctly:

Please make the following configuration changes for my ASP.NET application.

1) configure <path to dasblog> as an ASP.NET Application Directory.

2) give Read/Write (and Modify) permission for "ASP.NET Machine Account" and "NT AUTHORITY\NETWORK SERVICE" to:

  • <path to dasblog>/SiteConfig/
  • <path to dasblog>/content/
  • <path to dasblog>/logs/
